# Description
Ce projet vise à implémenter les opérations de lecture et d'écriture sur un système de fichiers FAT32 au sein d'un OS minimaliste. 
La base de code de l'OS est issue du travail de Philipp Oppermann ([Writing an OS in Rust](https://os.phil-opp.com/)).

# Fonctionalités
  - ls
  - cd
  - mkdir
  - touch
  - write
